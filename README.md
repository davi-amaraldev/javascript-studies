## 📖 Sobre o Repositório

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos durante minha formação em JavaScript.

O objetivo é construir uma base sólida na linguagem, estudando desde lógica de programação e fundamentos do JavaScript até conceitos mais avançados, como funções, arrays, objetos, protótipos, programação orientada a objetos, assincronismo e futuramente Node.js.

Este repositório também serve como registro da minha evolução prática, com códigos organizados por seções e aulas.

---

## 🎯 Objetivos

- Consolidar os fundamentos de programação
- Dominar JavaScript moderno
- Praticar lógica de programação
- Entender profundamente funções, arrays, objetos e protótipos
- Estudar Programação Orientada a Objetos com JavaScript
- Desenvolver uma base forte para Node.js, TypeScript e Back-End
- Criar um portfólio técnico consistente no GitHub
