## 📖 Sobre

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos durante um curso completo de JavaScript.

O objetivo é construir uma base sólida em desenvolvimento web, praticando desde conceitos fundamentais de programação até recursos avançados da linguagem.

Todo o conteúdo está organizado por seções, acompanhando minha evolução ao longo da formação.

---

## 🎯 Objetivos

- Consolidar os fundamentos de programação
- Dominar JavaScript moderno (ES6+)
- Aprender boas práticas de desenvolvimento
- Desenvolver projetos práticos
- Criar uma base sólida para Node.js e TypeScript
- Construir um portfólio técnico consistente
